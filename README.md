# MBRjun-Blog
MBRjun-Blog 的源码仓库  

这是一个使用 Hexo 搭建的 Blog，地址为：[mbrjun.cn](https://www.mbrjun.cn)  

出于性能和一些其他原因，该 Blog 已在 2022 从 Typecho 转移到 Hexo 平台  


## 构建
该 Blog 使用 GitHub Action 实现自动构建  

---

手动构建：  
1. 安装 Node.js v18 或 v19
2. 运行 ``make hexo`` 命令进行生产构建
